# ACME Release Notes Instructions

- Preserve product and technical vocabulary: microservice, GraphQL, gRPC,
  event bus, saga pattern, and circuit breaker.
- Use sections appropriate to the release: 🚀 New Features, 🔌 API Changes,
  ⚡ Performance, 🔒 Security, 🐛 Bug Fixes, and ⚙️ Technical.
- Do not mention commit hashes or internal ticket IDs.
- Write concise, customer-facing language.
